# React Components

We provide a set of React components that can be used to build a custom UI. The components are built using [React](https://reactjs.org/) and [tailwind](https://tailwindcss.com/).

## Storybook

You can view our storybook [here](https://storybook.iguhealth.app) that showcases the components and their usage.

## Source code

The source code for the components is available [here](https://github.com/IGUHealth/IGUHealth/tree/main/packages/components).
