# Local Development

Get started by Downloading our [repo](https://github.com/iguhealth/iguhealth) or setting up an account.

## What you'll need

- [Node.js](https://nodejs.org/en/download/) version 22.7.0 or above
- [Docker](https://www.docker.com/) version 25.0.3 or above

## Setup
1. <b>Install dependencies</b>
    
    Run the following command to install dependencies required for iguhealth. 
    From the root of the repository run the following command:
    ```bash
    sh install.sh
    ```

2. <b>Start the services</b>

    Run the following command to start services required for iguhealth. 
    From the root of the repository run the following command:
    ```bash
    docker compose -f docker/services-compose.yml up
    ```

3. <b>Start the server</b>
    
    Run the following command to start the iguhealth server. 
    From the root of the repository run the following command:
    ```bash
    yarn workspace @iguhealth/server watch
    ```

4. <b>Create a tenant</b>
    
    Run the following command to create a new tenant. 
    From the root of the repository run the following command:
    ```bash
    yarn workspace @iguhealth/server cli admin tenant create
    ```
    Follow the prompts to create a new tenant and user.

5. <b>Start the admin Application</b>
    
    Run the following to start the admin app:
    ```bash
    yarn workspace @iguhealth/admin-app start
    ```

6. <b>Login to the admin app</b>
   
   Go to `http://localhost:3000/auth/login` and follow the prompts to login to the admin app.
