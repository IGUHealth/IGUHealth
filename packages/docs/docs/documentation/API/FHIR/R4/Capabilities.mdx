# Capabilities

Capabilities returns what operations and resources are supported by the server. 
This is returned to the client as a [CapabilityStatment](/documentation/Data_Model/R4/CapabilityStatement) resource.


## API
<Tabs>
  <TabItem value="cli" label="CLI">
```bash
iguhealth api capabilities r4
```
  </TabItem>
  <TabItem value="curl" label="Curl">
```bash
 curl --request GET --header "Authorization: Header" --header "Content-Type: application/fhir+json" ${domain}/w/${tenant}/api/v1/fhir/r4/metadata
```
  </TabItem>
   <TabItem value="typescript" label="Typescript">
```typescript
await client.capabilities({}, FHIRVersion);
```
   </TabItem>
</Tabs>
