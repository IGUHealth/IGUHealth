# React Components
We provide a set of React components that can be used to build a custom UI. The components are available [here](https://storybook.iguhealth.app) and are built using [React](https://reactjs.org/) and [tailwind](https://tailwindcss.com/).

## Source code
The source code for the components is available [here](https://github.com/IGUHealth/IGUHealth/tree/main/packages/components).

## Installation
### NPM
```bash
npm install @iguhealth/components
```
### Yarn
```bash
yarn add @iguhealth/components
```
